  Query for events using the 'events/search' endpoint.
    Format is: event{param=value,...} 

    Example: event{type="WORKORDER",assetSubtrees=[12,345,6,7890]} 

  Templated variables can also be used with [[variable]] or $variable
    Example: event{assetSubtrees=[$Variable]}

  Filter on the queried events
    Format is: filter{property comparator value,...}
    Comparator can be =, !=, =~, !~ 

    Example: filter{metadata.key1=~".*test", subtype!="ABC"}
